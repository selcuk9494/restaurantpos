# Saha Kurulum Notu

Bu dosya, `device-agent` kurulumunu yapacak kisi icin en kisa operasyon ozetidir.

## 1. Hangi EXE

- Standart Windows cihaz: `release/win-x64/resto-device-agent-0.1.0-x64.exe`
- ARM Windows cihaz: `release/win-arm64/resto-device-agent-0.1.0-arm64.exe`

## 2. Nereye Kopyalanacak

- Onerilen klasor: `C:\RestoDeviceAgent\`

## 3. Ilk Acilista Doldurulacak Alanlar

1. `Izlenecek klasor`
2. `Basarili arsiv klasoru`
3. `Hatali arsiv klasoru`
4. `Sube adi`
5. `Terminal adi`
6. `Sube kodu`
7. `Terminal kodu`
8. `Aktif yazici eslesmeleri`

## 4. Ilk Islem

1. `Ayarlari Kaydet`
2. `Ayarlari Yedekle`
3. `Izlemeyi Baslat`

## 6. Basarili Kurulum Kriteri

- Dosya algilaniyor
- Dogru yaziciya gidiyor
- Arsive tasiniyor
- Hata klasorune dusmuyor

## 7. Sorun Olursa

1. `Canliya Hazirlik`
2. `Cihazlar > Tani`
3. `Cihazlar > Loglar`
4. `Terminal Kaydi > Son Uzak Cevap`
5. `Yedek ve Kurulum`
