# Hizli Kurulum Checklist

Bu sayfa yazdirilabilir kisa kontroldur. Her madde tamamlandikca isaretlenir.

## Paket

- [ ] Dogru EXE secildi (`x64` veya `arm64`)
- [ ] EXE `C:\RestoDeviceAgent\` altina kopyalandi
- [ ] Uygulama acildi

## Temel Ayarlar

- [ ] Izlenecek klasor secildi
- [ ] Basarili arsiv klasoru secildi
- [ ] Hatali arsiv klasoru secildi
- [ ] En az bir aktif yazici eslendi
- [ ] Sube adi girildi
- [ ] Terminal adi girildi
- [ ] Sube kodu girildi
- [ ] Terminal kodu girildi

## Ilk Kayit

- [ ] `Ayarlari Kaydet` calistirildi
- [ ] `Ayarlari Yedekle` ile ilk yedek alindi
- [ ] `Canliya Hazirlik` kartinda kritik eksik kalmadi

## Smoke Test

- [ ] `Izlemeyi Baslat` calistirildi
- [ ] [../../apps/device-agent/samples/MUTFAK-202604210930.1](../../apps/device-agent/samples/MUTFAK-202604210930.1) izlenen klasore kopyalandi
- [ ] Test dosyasi algilandi
- [ ] Dogru yaziciya gitti
- [ ] Arsive tasindi
- [ ] Hata klasorune dusmedi

## Ikinci Test

- [ ] [../../apps/device-agent/samples/BAR-202604210931.2](../../apps/device-agent/samples/BAR-202604210931.2) ile JSON akis dogrulandi
